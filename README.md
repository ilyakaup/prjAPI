## API к проекту Mestro: 

```
Примеры запросов:

GET localhost:3000/users                          : вернуть JSON-список всех пользователей
GET localhost:3000/cards                          : вернуть JSON-список всех карточек
GET localhost:3000/users/8340d0ec33270a25f2413b69 : вернуть JSON-пользователя с переданным после /users идентификатором.

```
